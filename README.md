#### book

本规范文档基于 `gitbook` 构建 `web`服务，主要是便于阅览。

###### 拉取镜像

```shell
# 此流程可以 ignore
docker pull alicfeng/gitbook:team_standard_rc
```

###### 运行规范文档服务

```shell
docker run -it --name team_standard -p 8088:2015  alicfeng/gitbook:team_standard_rc
```
